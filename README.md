# vietnam-cities-list
List of centrally controlled cities and provincial cities in Viet Nam extracted from https://en.wikipedia.org/wiki/List_of_cities_in_Vietnam

A city includes these information: 
- city
- province
- area
- population

Example:

```json
    {
        "city": "Hà Nội",
        "province": "Hà Nội",
        "area": "3,324.5",
        "population": "7,216,000"
    },
    {
        "city": "Nha Trang",
        "province": "Khánh Hòa",
        "area": "251.0",
        "population": "392,279"
    }
```
